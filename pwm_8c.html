  2. On top of that the PWM does NOT pick up any new values unless the counter reaches its end. So to fast pick up a new value you have to disable it and then enable it again.
  3. The output polarity and reverse polarity bits are effective also if the PWM is disabled but not of there is no clock

errStatus gpioPwmSetClockFreq ( uint32_t  clockFreq)

Sets the frequency desired for the PWM clock.

The desired frequency must be between 4688 Hz and 19.2 MHz. This is because the PWM is sourced from a 19.2 Mhz clock and has a 12 bit divider.

Parameters
clockFreqThe desired PWM clock frequency in Hertz.
Returns
An error from errStatus.

errStatus gpioPwmSetModes ( ePwmMode  mainMode,
ePwmSubMode  subMode 
)

Sets the mode for PWM.

There are two "main" modes for the PWM module which is set by mainMode. PWM mode has two available submodes which must be specified by subMode. If in serialiser mode subMode is not checked.

Parameters

subModeSets the submode when mainMode is PWM.
